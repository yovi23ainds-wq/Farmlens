# 🌱 FarmLens
### AI-Powered Smart Crop Disease Detection & Farmer Assistance Platform

FarmLens is an intelligent web-based application that helps farmers identify crop diseases instantly using Artificial Intelligence. By simply uploading an image of a plant leaf, users receive disease predictions along with treatment recommendations, helping improve crop health and productivity.

---

## 🚀 Features

- 🌿 AI-based Crop Disease Detection
- 📸 Upload Plant Images for Instant Analysis
- 🤖 Machine Learning Powered Predictions
- 💊 Disease Prevention & Treatment Suggestions
- 👤 User Authentication & Profile Management
- 📊 Clean and Responsive Dashboard
- 📱 Mobile-Friendly User Interface
- ⚡ Fast and Accurate Predictions

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yovi23ainds-wq/Farmlens1.git
```

### Navigate to Project

```bash
cd Farmlens1
```

---

## Install Frontend Dependencies

```bash
npm install
```

---

## Start Frontend

```bash
npm run dev
```

---

## Backend Setup

Navigate to the backend folder.

```bash
cd backend
```

Install Python dependencies.

```bash
pip install -r requirements.txt
```

Run the backend server.

```bash
python main.py
```

---

## 💡 How It Works

1. User signs in to the application.
2. Uploads a plant leaf image.
3. The image is processed using the trained AI model.
4. FarmLens predicts the disease.
5. Treatment and preventive measures are displayed.
6. Users can monitor and manage crop health efficiently.
